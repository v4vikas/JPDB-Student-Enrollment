                                  Student Enrollment Form using JSON POWER DATABASE

# Description :-
Student Enrollment Form Using Json Power Data Base in which user can add student information and update the information as per requirement there is also a option for reset in which if wrong information is entered then after clicking reset button all the information will be erased

4. Real-Time Database
5. Ajax Enabled
6. development without any server side scripting language or programming language
